## 🎨 Premium Aesthetics

Folio AI abandons the mundane forms of traditional resume builders. Leveraging **deep glassmorphism**, **custom micro-animations**, and **responsive layouts**, our builder makes career preparation visually stunning and exceptionally smooth in both **Light and Dark Mode**.

<br/>

## 🛠 Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Lucide Icons
- **Backend Architecture:** Node.js, Express (API routes ready for v2 integration)
- **Design System:** Custom CSS tokens with native CSS variables for seamless theme switching

<br/>

## 🚀 Getting Started

**1. Clone the repository**
\`\`\`bash
git clone https://github.com/your-username/folio-ai.git
\`\`\`

**2. Install Dependencies**
\`\`\`bash
cd folio-ai/frontend
npm install
\`\`\`

**3. Run the Development Server**
\`\`\`bash
npm run dev
\`\`\`
